# Политика конфиденциальности

**Приложение:** Судоку Классик
**Версия документа:** 1.0
**Дата:** 2026-05-24
**Контакт разработчика:** terekh-spb@mail.ru

## 1. Какие данные мы собираем

Само приложение **не собирает и не передаёт никакие персональные данные** о пользователе. Игра работает полностью на устройстве: прогресс, настройки звука/вибрации и активный уровень сохраняются в локальном хранилище (`localStorage` WebView) на вашем устройстве и никогда не покидают его.

## 2. Реклама

В приложении используется рекламная сеть **Яндекс Мобильная Реклама** (Yandex Mobile Ads SDK). Реклама показывается:

- Между прохождениями уровней (interstitial).
- По вашему явному запросу (rewarded — восстановление сердца после проигрыша).

Яндекс Мобильная Реклама может собирать и обрабатывать данные, связанные с показом рекламы (идентификатор устройства для рекламы, версию ОС, страну, грубое местоположение по IP). Подробности — в политике конфиденциальности Яндекса:

- https://yandex.ru/legal/confidential/
- https://yandex.ru/legal/mobileads_sdk_agreement/

## 3. Cookies / Tracking

Приложение не использует cookies (это не веб-приложение). Никакая аналитика (Firebase, Google Analytics, AppMetrica и т.п.) не подключена.

## 4. Хранение данных

Все игровые данные (прогресс, настройки) хранятся в локальном хранилище приложения на устройстве. При удалении приложения они удаляются вместе с ним. Никаких облачных синхронизаций нет.

## 5. Возрастные ограничения

Приложение не имеет возрастных ограничений по содержимому (0+).

## 6. Права пользователя

Вы можете:
- Удалить локальные данные через настройки Android (Приложения → Судоку Классик → Очистить данные).
- Удалить само приложение.
- Связаться с разработчиком по адресу **terekh-spb@mail.ru** по любым вопросам.

## 7. Изменения политики

Эта политика может обновляться. Актуальная версия всегда доступна по тому же URL.
