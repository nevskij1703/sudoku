# Судоку Классик

Классическое судоку 9×9 для Android. HTML/JS-проект, упаковываемый в APK через [html2apk](https://github.com/anthropics/html2apk) для публикации в РуСтор.

## Геймплей

- Уровни идут один за другим, между ними — экран выбора **сложности** (Простой / Средний / Сложный) и **режима** (пока только «Классика», в будущем добавятся варианты).
- На каждый уровень даётся **3 сердечка** и **5 подсказок**.
- Если поставил неверную цифру — цифра подсвечивается красным, отнимается 1 сердце.
- Когда сердца кончились — можно посмотреть rewarded-рекламу для +1 ♥ или начать новый уровень.
- Помощник-подсветка автоматически выделяет строку, столбец, блок 3×3 выбранной ячейки + все одинаковые цифры на поле.

## Инструменты на игровом экране

- Цифры **1–9** — вписать цифру (или поставить заметку, если активен карандаш).
- **✏ Карандаш** — режим заметок: повторный тап по цифре в одной ячейке добавляет/убирает её.
- **⌫ Ластик** — очистить ячейку (цифру и заметки).
- **💡 Подсказка** — открыть правильную цифру в выбранной ячейке (5 на уровень).
- **↶ Отмена** — откатить последнее действие (до 50 шагов).
- **⏸ Пауза** — поставить таймер на паузу, открыть настройки.

## Dev-режим

Открой `?dev=1` в URL (`http://localhost:8776/?dev=1`). Появится правый верхний оверлей с кнопками:

- Регенерировать уровень с заданной сложностью.
- Перезапустить текущий уровень.
- Заполнить решением (instant win — для проверки финального экрана).
- Подкинуть сердце / подсказку.
- Показать абсолютный score текущего уровня и список применённых техник.
- Сгенерить 50 уровней со статистикой (для калибровки порогов сложности).
- Проверить uniqueness 50 уровней.
- Toggle Mock Ads.
- Сбросить localStorage.

Весь dev-код обёрнут в `HTML2APK:DEV_ONLY` маркеры — html2apk при `-Release` сборке вырежет его автоматически.


И открой `http://localhost:8776/`. Preview port `8776` зафиксирован в `.claude/launch.json` (8771–8775 заняты другими проектами мастерской).

## Сборка APK

```powershell
& "$env:LOCALAPPDATA\Programs\html2apk\html2apk.ps1" `
  -ProjectFolder "C:\Users\Александр\Desktop\Claude\06_Sudoku" `
  -OutputFile "$env:USERPROFILE\Downloads\SudokuClassic.apk" `
  -YandexAdsBridge `
  -RuStoreReviewSdk
```

`appName` / `appId` / `iconPath` / `orientation` / `immersive` / `statusBarStyle` подхватываются из `.claude/build-config.json`. Для release-сборки добавь `-Release` — keystore из `Store_Info/keystore/` будет использован автоматически.

Skill `prepare-release-candidate` делает всё это за вас + проверки миграций сейва + интерактивное подтверждение.

## Структура проекта

```
js (в корне)         — classic IIFE-модули, грузятся в порядке из index.html
docs/                — техдока (сейвы, реклама, генератор)
Store_Info/          — метаданные для публикации (keystore, иконки, политика)
.claude/             — конфиги для Claude Code (build-config, launch, release-state)
```

JS-файлы по ролям:

| Файл | Роль |
|---|---|
| `config.js` | константы (ads, генератор, баланс) |
| `migrations.js` | реестр миграций сейва |
| `storage.js` | localStorage обёртка |
| `sudokuCore.js` | солвер + ClassicVariant |
| `sudokuTechniques.js` | human-style техники для оценки сложности |
| `sudokuGenerator.js` | генератор уровней + rateDifficulty |
| `ads.js` | Yandex Mobile Ads bridge / mock |
| `rustoreReview.js` | RuStore in-app review SDK |
| `audio.js` | звуки и вибрация |
| `ui.js` | screens / modals |
| `board.js` | рендер 9×9 grid + подсветка |
| `numberPad.js` | нижняя панель ввода |
| `game.js` | state machine текущего уровня |
| `main.js` | bootstrap и склейка |
| `devPanel.js` | dev-инструменты (вырезается release-сборкой) |

Подробности — в [docs/](docs/).
